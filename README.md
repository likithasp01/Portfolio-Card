## Personal Portfolio Card

* A simple and elegant Personal Portfolio Card Website built using HTML, CSS, and JavaScript.

* This project showcases a profile card with skills, contact button, CV download button, animations, and modern UI styling.

## Features

* Profile Image with animation

* Name and Education Details

* Short Professional Description

🏷 Skills Tags with hover effect

* Contact Button (JavaScript alert)

* Download CV Button

* Smooth animations using CSS keyframes

* Modern card UI with shadows and hover effects

## Technologies Used

* HTML5

* CSS3 (Flexbox, Animations, Shadows, Hover Effects)

* JavaScript (Event Listeners & DOM Manipulation)
